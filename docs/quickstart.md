# Quick Start

Get started with the AI-Humanoid Technical Book in minutes.

## Prerequisites

- **Node.js**: Version 18 or higher
- **npm or yarn**: Package manager for Node.js
- **Git**: Version control system

## Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Ai_Humanoid_Book
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

The documentation site will open in your browser at `http://localhost:3000`.

## Explore the ROS 2 Module

The main content of this book is organized in the "ROS 2 as Robotic Nervous System" module:

1. **[ROS 2 Fundamentals](./ros2-nervous-system/ros2-fundamentals)** - Learn the core concepts of ROS 2 architecture
2. **[ROS 2 Communication with Python Agents](./ros2-nervous-system/python-agents)** - Connect AI agents to robot controllers
3. **[Humanoid Modeling with URDF](./ros2-nervous-system/urdf-modeling)** - Understand robot structure with URDF
4. **[Glossary](./ros2-nervous-system/glossary)** - Reference for key ROS 2 terms
